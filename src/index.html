프론트엔드 개발자 성격 유형 테스트 👀

1. 우테코 미션을 하면서 타입스크립트를 사용하게되는 순간 "아 살겠다"라는 기분이 들었다.

2. CSS 작성시 1px 단위로 신경쓴다.

3. 버그가 생기면 디버깅할 생각에 군침이 싹돈다.

4. 피그마,프로그래밍 요구사항에 없는 UI요소를 추가하고싶은 충동이 자주 든다.

5. 개발하다가 "나는 천재인가?"라는 사색에 잠기곤한다.

6. 이번 페이먼츠 미션 소개 페이지에 기능 구현부터 먼저하고 UX는 피드백 받고나서 하라고 되있지만 일단 UX 요소 당장 추가할거다.

7. 타입 에러 불끄는대는 'any'가 최고라고 생각하고 애정한다.

8. 영화 미션에서 반응형 CSS를 구현해야했을 때 너무 행복했다.

9. 새로운 라이브러리를 사용할 때 일단 공식문서부터 정독한다.
